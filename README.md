# mpx-project

> A mpx project

## Dev

```bash
# install dep
npm i

# for web-client  dev
npm run watch:ssr-client

# for web-server dev
npm run watch:ssr-server
# for server dev
node server.js
```

## Prod
```bash
# install dep
npm i

# for web-client  prod
npm run build:ssr-client

# for web-server prod
npm run build:ssr-server
# for server dev
node server.js
```
